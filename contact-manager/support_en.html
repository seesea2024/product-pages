App Features

ContactManager is a professional address book management tool offering the following core features:

Duplicate Contact Removal

Intelligently detect and merge duplicate contacts, supporting detection by phone number or name.

Contact Management

Easily modify, delete, and edit contact information with support for batch operations.

Contact Merging

Smart merging of duplicate contacts, automatically retaining the most complete information to prevent data loss.

Import & Export

Import contacts from the system address book and export them in VCF format for backup or sharing.

System Sync

Sync contacts within the app back to the system address book, utilizing iCloud for cross-device synchronization.

Backup & Restore

Create local backup files to restore address book data at any time, with support for backup history management.

Privacy & Security

Data security is our most important commitment:

  • Purely Local Operation: All contact data processing, modification, merging, and deletion are completed locally on your device.
  • No Cloud Uploads: The app does not upload your address book data to any third-party servers.
  • System Synchronization: Cross-network transmission is only supported via the iOS system framework calling iCloud services for system address book synchronization.
  • Data Control: You have complete control over your data and can delete or export it at any time.
  • Transparent Permissions: The app only requests access to your contacts when necessary, with clear and explicit purposes.

FAQ

Q: Does the app collect my contact information?

A: No. The app only processes address book data locally on your device and never uploads it to any server.

Q: What fields are included when importing contacts from the system address book to ContactManager?

A: The fields managed in ContactManager include basic information such as phone numbers, names, emails, company, and job titles. ContactManager is a simplified version of the system address book, not a replacement. File-related fields like images or videos not supported

Q: Will merging contacts cause data loss?

A: No. The merging rules are based on phone numbers and names. The app will not delete any contact data before confirmed by the user.

Q: Where are the backup files stored?

A: Backup files are stored in the app's local document directory and are not uploaded to the cloud (unless you sync them manually via iCloud).

Q: How do I export contacts?

A: Select the "Export" feature in the app to export contacts as VCF files, which can then be shared via AirDrop, email, etc.
