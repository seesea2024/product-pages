Contact Cleaner Lite

Smart Contact Management Tool

App Overview

Contact Cleaner Lite is a professional iOS contact management application designed to help users clean up and manage their address book contacts. All operations are performed locally on your device to ensure data security.

Core Features:

  • Smart Duplicate Detection: Automatically detect duplicate contacts in your address book, supporting detection by phone number and name
  • Contact Management: Easily modify, delete, and merge contacts
  • Import & Export: Import contacts from system address book, export contacts in VCF format
  • System Sync: Sync app contacts back to system address book (iCloud-based)
  • Backup & Restore: Create local backup files, restore address book data anytime

Privacy Policy

1. Data Collection

Contact Cleaner Lite follows strict privacy protection principles:

  • We do not collect any personal information or upload your address book data to any third-party servers
  • The app processes contact data only locally on your device
  • All contact data remains under your control and is always stored on your device

2. Data Storage

  • Contact data read from your system address book is stored only locally on your device
  • Backup files are saved in the app's documents directory and are not uploaded to the cloud (unless you manually sync via iCloud)
  • The app does not use any third-party cloud services or databases

3. Data Transmission

  • Only network transmission: Using iOS system framework to call iCloud services for system address book synchronization
  • The app itself does not actively connect to the internet for data transmission
  • When exporting VCF files via the "Share" feature, the system share menu is used, and data transmission is controlled by the system

4. Data Security

  • The app uses iOS-provided security framework to access contact data
  • All data transfers are performed through iOS system frameworks and protected by system security mechanisms
  • The app does not store any passwords, account information, or other sensitive data

5. User Rights

You have complete control over your data:

  • Access Anytime: View and manage all address book contacts in the app at any time
  • Export Backup: Export contacts in VCF format for backup
  • Uninstall Clears: When uninstalling the app, all data in the app is automatically cleared
  • Permission Control: You can revoke the app's contact access permission in iOS settings at any time

Data Source Explanation

Contact Cleaner Lite's contact data comes from:

  • System Address Book: Reads your system address book data through the iOS Contacts framework
  • VCF Files: Imports contacts from VCF format files through the file import feature

Important Note: The app does not collect or obtain contact information from any other sources.
