常见问题

资产如何截屏？

各渠道资产页面截屏路径：

  • 支付宝：首页 → 我的 → 总资产
  • 微信：首页 → 我 → 服务 → 理财通 → 我的资产
  • 农行：首页 → 我的 → 资产负债
  • 招行：首页 → 账户总览
  • 中行：首页 → 我的 → 资产 → 资产管理
  • 工行：首页 → 我的 → 我的资产负债
  • 平安证券：首页 → 我的 → 我的持仓 → 我的资产明细

截屏提示：请确保截屏包含完整的资产数值。

如何导出银行流水？

各渠道导出路径：

  • 支付宝：我的 → 账单 → 右上角 ⋯ → 开具交易流水证明
  • 微信：我 → 服务 → 钱包 → 账单 → 右上角 ⋯ → 下载账单
  • 农行：首屏 → 搜索「流水打印」
  • 招行：首屏 → 搜索「流水打印」
  • 中行：首屏 → 搜索「交易流水打印」
  • 工行：首屏 → 搜索「历史明细打印」

注意：下载后的流水文件可能会压缩或加密，需要根据渠道提供的密码解密、解压。对于 PDF 加密文件，可以打开后另存为无密码 PDF 再导入。

如何导入银行流水？

在「流水账」页面点击右上角菜单，选择「导入流水」，然后选择对应的银行和文件格式即可自动导入。

支持哪些银行或渠道？

目前支持招商银行、中国银行、工商银行、农业银行、支付宝、微信、平安证券等。

数据会丢失吗？

数据保存在本机，建议定期通过 iCloud 备份。

如何删除导入的数据？

在「流水账」页面使用筛选功能找到对应银行的数据，或通过「导入」页面的银行管理功能删除。

图表数据不准确？

请检查日期范围设置和筛选条件。图表显示的是筛选后数据的汇总结果。
