常见问题

如何截屏各个渠道资产图片

各个渠道资产图片截屏路径如下：

  1. 支付宝：首页 -> 我的 -> 总资产
  2. 微信：首页 -> 我 -> 服务 -> 理财通 -> 我的
  3. 招行：首页 -> 账户总览
  4. 中行：首页 -> 我的 -> 我的资产（点击资产数字）-> 资产管理
  5. 工行：首页 -> 我的（我的资产负债）
  6. 农行：首页 -> 我的 -> 资产负债（点击我的资产金额） -> 资产负债
  7. 平安证券：首页 -> 我的 -> 我的持仓 -> 我的资产明细

如何获取银行流水？

登陆银行手机应用，首页搜索”流水打印“或者”交易流水打印“，选择时间区间和录入邮箱后，PDF流水文件会发送到用户邮箱。注意文件一般会加密，需要根据银行提供的解密数字进行解密。如果是直接加密PDF可以打开PDF文件后选择打印另存为新的PDF文件，得到的就是不需要密码的PDF文件。

如何导入银行流水？

在「流水账」页面点击「导入」按钮，选择对应的银行，然后选择PDF或CSV格式的账单文件即可自动导入。

支持哪些银行或者渠道？

资产图片解析目前支持支付宝、微信、招商银行、中国银行、工商银行、农业银行和平安证券；PDF流水解析支持招商银行、中国银行和农业银行。后续版本将增加更多银行支持。

数据会丢失吗？

数据保存在本机，建议定期通过iCloud备份。升级到专业版可启用自动iCloud同步功能。

如何删除导入的数据？

在「流水账」页面使用筛选功能找到对应银行的数据，或通过「导入」页面的银行管理功能删除。

图表数据不准确？

请检查日期范围设置和筛选条件。图表显示的是筛选后数据的汇总结果。
